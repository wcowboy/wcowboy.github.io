JavaGui 02 Pannel-面板

  • Java
  view

在javaGui中，Panel是一个轻量级的容器类

1.Pannel是什么

在javaGui中，Panel是一个轻量级的容器类，可以将panel看作是画板。

2.怎么用panel

  1. Frame对象
//实例化Frame对象
        Frame frame = new Frame();
  1. Panel对象
//实例化一个Panel对象
        Panel panel = new Panel();

  1. 设置布局

    布局包括常见的流式布局,盒式布局等等

   //布局设置为空。具体布局后面再叙述
           frame.setLayout(null);
  1. 设置frame长宽,坐标
//设置frame的坐标以及大小，等价于：
        //frame.setLocation(100, 100);//设置坐标
        //frame.setSize(500, 500);//设置长款（大小）
        frame.setBounds(100,100,500,500);
  1. 背景颜色

设置背景颜色的参数是Color,颜色类,直接new一个传进去,Color接受RGB和RGBA,也有自己常用的常量颜色

        //设置背景颜色
        frame.setBackground(new Color(92, 127, 103));
  1. 设置pannel坐标

注意,panel的坐标是以Frame为参考系,而frame是以屏幕为参考系

        //panel坐标，相对于frame
        panel.setBounds(50,50,400,400);
  1. panel颜色
        //panel颜色
        panel.setBackground(new Color(142, 115, 104));
  1. frame添加panel

add方法参数为Component对像,Panel继承于Container,Container继承于Component,因此可以直接将Panel对象传进去

        //frame.add
        frame.add(panel);
  1. 设置frame可见
        frame.setVisible(true);
  1. 关闭窗口按钮事件重写

这里是适配器模式.只需要重写我们需要重写的方法即可

        //监听窗口关闭
        //设计模式-适配器模式
        frame.addWindowListener(new WindowAdapter() {
            @Override
            public void windowClosing(WindowEvent e) {
                super.windowClosing(e);
                System.exit(0);
            }
        });

3.完整代码&代码结果

import java.awt.*;
import java.awt.event.WindowAdapter;
import java.awt.event.WindowEvent;

//Panel，可以看成一个空间，但是不能单独存在，这里添加到Frame上面
public class TestPanel {
    public static void main(String[] args) {
        //实例化Frame对象
        Frame frame = new Frame();

        //实例化一个Panel对象
        Panel panel = new Panel();

        //布局设置为空。具体布局后面再叙述
        frame.setLayout(null);

        //设置frame的坐标以及大小，等价于：
        //frame.setLocation(100, 100);//设置坐标
        //frame.setSize(500, 500);//设置长款（大小）
        frame.setBounds(100,100,500,500);

        //设置背景颜色
        frame.setBackground(new Color(92, 127, 103));

        //panel坐标，相对于frame
        panel.setBounds(50,50,400,400);

        //panel颜色
        panel.setBackground(new Color(142, 115, 104));

        //frame.add
        frame.add(panel);

        frame.setVisible(true);

        //监听窗口关闭
        //设计模式-适配器模式
        frame.addWindowListener(new WindowAdapter() {
            @Override
            public void windowClosing(WindowEvent e) {
                super.windowClosing(e);
                System.exit(0);
            }
        });
    }
}
