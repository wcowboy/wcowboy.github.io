JavaGui05 Listener-监听

JavaGui中简单的监听

窗口监听

在前面的学习中，我们的窗口点击右上角的X是不能关闭的，这怎么能行，今天我们就来赋予关闭的功能，来看下面这个代码

public class TestFrame {
    public static void main(String[] args) {
        Frame frame = new Frame("窗口关闭");

        //需要设置可见性 w,h
        frame.setVisible(true);

        //设置窗口大小
        frame.setSize(400,400);

        //设置背景颜色
        frame.setBackground(new Color(87, 94, 72));

        //弹出基本位置/
        frame.setLocation(200,200);

        //设置大小固定
        frame.setResizable(false);
    }
}

会发现，这个代码运行起来是不能直接关闭的，因此，我们得想办法给他关闭，这里就得提到监听了，监听器可以监听你设置的组件的行为，比如点击关闭按钮，触发监听器

给fram添加一个WindowListener,因为fram是一个窗口，所以用WindowListener,new一个WindowAdapter,这里传入的应该是要WindowListener，为什么现在new WindowAdapter呢？因为WindowListener需要重写里面的所有方法，像下面一样，因此我，我们传一个适配器。需要哪个重写哪个。

最后重写windowClosing方法就行了，像这样

frame.addWindowListener(new WindowAdapter() {
            @Override
            public void windowClosing(WindowEvent e) {
                System.exit(0);
            }
        });

动作监听

说完了窗口监听，下面来看看事件监听

public class TestButton {
    public static void main(String[] args) {
        Frame frame = new Frame("事件监听");
        frame.setBounds(100,100,300,100);

        Button btn = new Button("btn");

        frame.add(btn);

        frame.setVisible(true);

        btn.addActionListener(new ActionListener() {
            @Override
            public void actionPerformed(ActionEvent e) {
                System.out.println("btn 被点击");
            }
        });
    }
}

这么看来，其实和窗口监听事件逻辑差不多

更优雅的代码

我们可以将关闭，封装成一个方法，监听器自己写一个，就像这样

public class TestActionEvent {
    public static void main(String[] args) {
        //按下按钮触发事件
        Frame frame = new Frame();

        Button button = new Button();

        //需要一个ActionListener
        MyActionListener myActionListener = new MyActionListener();
        button.addActionListener(myActionListener);

        frame.add(button, BorderLayout.CENTER);
        frame.pack();
        frame.setVisible(true);
        windowClose(frame);

    }

    //关闭窗口
    private static void windowClose(Frame frame){
        frame.addWindowListener(new WindowAdapter() {
            @Override
            public void windowClosing(WindowEvent e) {
                System.exit(0);
            }
        });
    }
}

//事件监听
class MyActionListener implements ActionListener{

    @Override
    public void actionPerformed(ActionEvent e) {
        System.out.println(123);
    }
}

两个组件公用一个监听器

上面的代码将监听器用自己的类实现了。那一个监听器可以监听多个组件嘛？答案是可以的

//两个按钮公用一个方法
public class TestActionTwo {
    public static void main(String[] args) {
        Frame frame = new Frame();

//        frame.setBounds(100,100,600,400);
        frame.setVisible(true);

        Button btn1 = new Button("btn1");
        Button btn2 = new Button("btn2");


        MyListener myListener = new MyListener();

        btn1.addActionListener(myListener);
        btn2.addActionListener(myListener);

        frame.add(btn1,BorderLayout.EAST);
        frame.add(btn2,BorderLayout.WEST);

        frame.pack();
    }
}

class MyListener implements ActionListener{
    @Override
    public void actionPerformed(ActionEvent e) {
        System.out.println( e.getActionCommand());

    }
}

如何布局

第一种想法

  1. 如图，整体可以堪称两行一列的表格布局，然后上面的布局里面是是边框布局，上面的中间看作一个两行一列的表格布局，下面的整体也一样

  2. 具体的代码实现如下

    public class Test02 {
    public static void main(String[] args) {
        //new一个Frame
        //设置布局为2行一列的表格
        //设置背景色
        //设置大小
        //设置位置
        //设置可见
        Frame frame = new Frame();
        frame.setLayout(new GridLayout(2,1));
        frame.setBackground(Color.black);
        frame.setSize(600,400);
        frame.setLocation(100,100);
        frame.setVisible(true);

        //上面
        Panel panelUp = new Panel(new BorderLayout());
        //上面左右两个button
        panelUp.add(new Button("btn-02"),BorderLayout.EAST);
        panelUp.add(new Button("btn-01"),BorderLayout.WEST);
        //上面中间的两个并排的,放在一个panel上
        Panel panelUpItem = new Panel(new GridLayout(2, 1));
        panelUpItem.add(new Button("btn-03"));
        panelUpItem.add(new Button("btn-04"));
        //panelUpItem添加到panelUp
        panelUp.add(panelUpItem);

        //下面
        Panel panelDown = new Panel(new BorderLayout());
        //下面左右两个button
        panelDown.add(new Button("btn-09"),BorderLayout.EAST);
        panelDown.add(new Button("btn-10"),BorderLayout.WEST);
        //中间2x2
        Panel panelDownItem = new Panel(new GridLayout(2, 2));
        for (int i = 0; i < 4; i++) {
            panelDownItem.add(new Button("btn-0" + (i + 5)));
        }
        //panelDown添加到panelDownItem
        panelDown.add(panelDownItem);

        //将两个panel添加到frame
        frame.add(panelUp);
        frame.add(panelDown);


   }

}
